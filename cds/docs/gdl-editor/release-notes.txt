GDL Editor - Release Notes

2013-03-11 - v0.9

+ Supports:
	* GDL specification v0.9 (http://www.openehr.org/releases/currentbaseline/proposals/GDL/guideline_definition_language_v0.12.pdf)
	* Archetype Definition Language (ADL) v1.4 and OpenEHR Template (OET) specifications (release 1.0.2, http://www.openehr.org/releases/1.0.2)
	* CSV terminologies 
	* OWL ontologies (without queries, yet)
+ First implementations for CDS form generator, HTML generator and Drools converter 
+ Language translations for English and Spanish
+ Current time configuration
